# Задачи

### Теоретична задачка
За всеки ред, кажете дали ще се изпълни или ще даде грешка. Ако се изпълни, какво ще изведе на конзолата?  
```cpp
template <typename T>
void func(T a, T b)
{
	std::cout << a << b;
}

int main()
{
	func<int>(1, 2);        //1
	func(1, 2);             //2
	func<double>(1.5, 2);   //3
	func(1.2, 2);           //4
	func(3, 'a');           //5
	func<int>(3, 'a');      //6
	func<int>(3, 010);      //7
}
```
